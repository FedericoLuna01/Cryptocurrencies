# How is my crypto

## Sobre la aplicación

Este proyecto se trata de una pagina web donde se consume la api de CoinGecko y se puede ver un poco de información sobre las cryptomonedas mas populares.

Esta basado en un video de YouTube. Para el diseño se uso MaterialUI y un paquete npm para el carousel.

[Sitio web](https://howismycrypto.netlify.app/)

[Repositorio](https://github.com/FedericoLuna01/Cryptocurrencies)

## Getting Started

Para tener una copia local de esta aplicación seguí los siguientes pasos.

### Requisitos previos

Instalar la ultima versión de NodeJS.

- npm
  ```sh
  npm install npm@latest -g
  ```

### Instalación

1. Clonar el repositorio.
   ```sh
   git clone https://github.com/FedericoLuna01/Cryptocurrencies
   ```
2. Instalar los paquetes de NPM.
   ```sh
   npm install
   ```
